# 🕷️ W.E.B – Spiderman Anomaly Detection System

Real-time anomaly detection system built on NVIDIA Jetson using computer vision and a custom cyberpunk-style dashboard.

## 🚀 Features
- Real-time object detection (Jetson inference)
- Forbidden objects detection: phone, wine glass, wine bottle, cup, remote, laptop
- Person count violation (> 2)
- Live dashboard (Spiderman UI)
- Popup alerts on anomalies
- CSV logging

## ▶️ Run

python3 venu_ano_detect_final.py

Then open:
http://localhost:5000

## 📁 Output
anomaly_log.csv (auto generated)

## 🛠 Tech Stack
Python, Flask, Jetson Inference, OpenCV
